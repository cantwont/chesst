# Chesst

A Python program designated to mass-finding [Chess.com](https://chess.com/) email hints along with their display names, for some fun :D.

Example output:
```
[name]: [email hint (if found)] | [display name (if found)]
```
Example names.txt:
```
MagnusCarlsen
Hikaru
Username
```
